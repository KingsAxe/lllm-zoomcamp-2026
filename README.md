## LLM Zoomcamp 2026 Homework

This repository contains my work for the LLM Zoomcamp 2026 Agentic RAG homework.

The project uses Qwen through an OpenAI-compatible API, a local Jupyter notebook workflow, and a Python environment isolated in `llmz/`.

Current focus:
- fetch course lesson markdown files from the course repository
- build retrieval and RAG pipelines over the lesson content
- extend the workflow into an agentic search loop for the homework questions
